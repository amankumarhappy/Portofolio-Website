## Run locally

```bash
# Clone the repo
git clone https://github.com/amankumarhappy/Portofolio-Website.git

# Move into the folder
cd Portofolio-Website

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open `http://localhost:3000` in your browser.
